# 八千代松陰文化祭ウェブサイト 2022 年度

一時的に公開してます。

## 使い方

環境変数を設定してから実行してください

```bash
npm build & npm start
```

## 環境変数

以下を設定してください

- ALGOLIA_ADMIN_API_KEY
- NEXT_PUBLIC_ALGOLIA_SEARCH_API_KEY
- NEXT_PUBLIC_ALGOLIA_INDEX
- NEXT_PUBLIC_ALGOLIA_APPLICATION_ID
